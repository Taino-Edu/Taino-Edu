👾 Sobre mim (o estritamente necessário)
Sou o Taino. Escrevo código em linguagens que brigam comigo antes do PM brigar, e desenho jogos onde o jogador perde mesmo quando acha que ganhou. Se a vida tivesse borrow checker, eu estaria inteiro.

🦀 Backend em Rust — Auxiliar de Programador na Transat. Tipagem forte, opiniões mais fortes ainda.
🎮 Mindpunk Lab — fundador. Fazendo jogos táticos de sobrevivência (WARR, Zona de Contenção, Fragmento 1) onde o tutorial é: boa sorte.
🐍 Python & Scraping — projetos paralelos tipo raspagem do Cartola IA, porque dado livre é dado meu.
🐧 Arch Linux — não porque é mais rápido, mas porque gosto de pagar o aluguel do meu próprio sistema operacional em commits de config.


Gerencio complexidade no servidor, gerencio o inventário do NPC, e no fim do dia gerencio o backup do meu ~/.config antes de quebrar tudo de novo.
